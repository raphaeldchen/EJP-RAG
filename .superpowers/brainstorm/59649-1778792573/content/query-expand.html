A — Accordion (expand in-place)

Clicking a query expands it inline below the list item. Other queries stay visible. Click again to collapse. Multiple queries can be open simultaneously.

Pros

  • Context preserved — see which query you're in
  • Can compare two queries at once
  • Familiar pattern

Cons

  • Panel gets tall fast with many labels
  • Lots of scrolling in a narrow column

B — Drill-down (replace panel view)

Clicking a query replaces the query list with a full candidate detail view. A "← Back" button returns to the list. The panel is always showing one thing at a time.

Pros

  • Candidates get full panel width
  • Cleaner — no nested scrolling
  • Easier to read scores & notes

Cons

  • Can't see list & detail simultaneously
  • One extra click to switch queries
