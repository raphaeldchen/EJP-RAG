A — Modal dialog (@st.dialog)

Clicking "Feedback History" opens a centered overlay. Main content is dimmed behind it. User closes it to return to searching. Clean, no layout changes needed.

Pros

  • Zero layout impact on main search
  • Native Streamlit primitive
  • Simple to implement

Cons

  • Centered, not right-sided (Streamlit limitation)
  • Can't see search results & history at same time

B — Inline right panel (columns)

Clicking toggles a right-side panel. The main content area shrinks to share the page. History is visible alongside the search area. Closes with an ✕ button.

Pros

  • Closest to "panel from the right" feel
  • Can reference history while searching

Cons

  • Main content area gets squeezed
  • More layout wiring needed
  • Search results harder to scan when narrow
